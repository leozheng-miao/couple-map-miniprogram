# 测试清单

## 微信开发者工具

- 新用户进入启动页。
- 新用户可创建情侣空间。
- 创建后自动进入地图页。
- 第二个用户可通过邀请码加入。
- 两个用户能看到同一批地点。
- 未加入空间的用户不能访问空间数据。
- 没有腾讯位置服务 Key 时，手动选点可用。
- 配置腾讯位置服务 Key 后，POI 搜索可返回地点。
- 新增餐厅后地图出现标记。
- 新增酒店后地图出现标记。
- 新增景点后地图出现标记。
- 地点详情能展示封面图。
- 地点详情能展示照片墙。
- 地点详情能展示正文。
- 可以新增文字便利贴。
- 可以新增带照片的便利贴。
- 可以编辑地点。
- 可以软删除地点。
- 删除地点后地图和列表不再显示。
- 相册页能聚合所有地点照片。
- 相册点击照片进入对应地点详情。
- 我的页面能复制邀请码。
- 我的页面能刷新邀请码。
- 我的页面能复制导出数据。

## 真机预览

- 创建空间流程可跑通。
- 邀请码加入流程可跑通。
- 新增地点可通过搜索或手动点选地图完成，不触发定位权限弹窗。
- 图片选择和上传正常。
- 地点保存后能重新进入查看。
- 便利贴保存后能重新进入查看。
- Tab 切换无白屏。

## 视觉验收

- 整体符合 `docs/assets/design-theme-reference.png`。
- 使用暖米色纸张背景。
- 使用红色主按钮。
- 地点卡片有手账感。
- 新增地点页有信封边框感。
- 便利贴使用暖黄、浅粉、浅绿、浅蓝。
- 图片展示优先，不被过多文字遮挡。
